# Feedback Workshop Registration Portal

A standalone registration portal for feedback workshop sessions across Netflix, Netflix Animation Studio, and Eyeline.

## Quick Deploy to Render

1. Fork or push this repo to your GitHub
2. Go to [render.com](https://render.com) → New + → Web Service
3. Connect your GitHub repo
4. Render auto-detects Node.js — just click Deploy
5. Your portal is live!

## Local Development

```bash
npm install
node server.js
```

Open http://localhost:3000

## Admin Access

Default admin code: `admin2026` (change in Settings after first login)
